# Slack Voice

Use this when writing Slack messages for Prasanth.

## Style

- Casual, direct, lowercase-first. Capitalise only proper nouns, acronyms, or emphatic laughter.
- Short messages beat polished paragraphs. Split multi-part thoughts across separate lines or messages.
- Start mid-thought. No greetings, sign-offs, preambles, or corporate filler.
- Use blunt but human phrasing. Push back with a short challenge when needed.
- Warmth comes from natural phrasing, not pleasantries.
- Minimal punctuation. Question marks are optional for rhetorical asks. Never use em dashes or semicolons.
- Use bullets only for status or coordination. Prefer `thing: action` bullets.
- Avoid bold, italics, title-case headers, and essay structure.

## Language

- Common particles and casual words: `leh`, `lah`, `sia`, `bro`, `sir`, `hmm`, `ah`, `cos`, `gonna`, `wanna`.
- Common short replies: `same`, `yes`, `ah ok`, `noted`, `LOL`, `hahaha`.
- Strategy language should stay concrete: `success metrics`, `working model`, `feedback`, `risk`, `next step`.
- For disagreement, state the position first, then the reason. Name the actual worry.

## Patterns

- Status:
  `done with the cleanup`
  `• dashboard: archived`
  `• pipeline: fixed`
  `• follow-up: with me`

- Pushback:
  `hmm i don't really buy that leh`
  `if the metric isn't moving then whats the point`

- Coordination:
  `who's joining`
  `i can only stay 30min`
  `ah ok found it`
